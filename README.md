# 💎 HireLens AI - Next-Gen Resume & LinkedIn Optimizer

HireLens AI is a premium, enterprise-grade analysis tool designed to bridge the gap between job seekers and recruiters using the power of Google's **Gemini 3.0** AI. 

![HireLens AI Preview](https://picsum.photos/seed/tech/1200/600)

## 🚀 Vision
In today's competitive job market, getting past the ATS (Applicant Tracking System) is 70% of the battle. HireLens AI provides users with the "Recruiter's Perspective," offering deep insights into their professional documents and social presence.

## ✨ Core Features
- 📄 **Multi-Format Extraction**: Seamlessly analyze `.pdf`, `.docx`, and `.txt` files.
- 🔗 **LinkedIn Profile Audit**: Optimize headlines and "About" sections for maximum visibility.
- 🎯 **JD Match Intelligence**: Calculate real-time compatibility scores with specific job descriptions.
- 💡 **AI Strategic Roadmap**: Actionable suggestions beyond just grammar—focusing on impact and growth.
- 🎤 **Predicted Interview Prep**: AI-generated questions tailored to your specific experience.
- 🌍 **Language Detection**: Support for global professional markets.
- 🎨 **Premium UI**: Immersive dark mode with dynamic mesh gradients and glassmorphism.

## 🛠️ Tech Stack
- **Frontend**: React 19, Vite, Tailwind CSS 4.0, Frame Motion.
- **Backend**: Node.js, Express, Multer (File Handling).
- **AI Engine**: Google Gemini 3.0 (Flash & Pro).
- **Icons**: Lucide React.
- **Styling**: Modern Design System with Mesh Gradients.

## 📦 Installation & Setup

1. **Clone the repository**:
   ```bash
   git clone https://github.com/YOUR_USERNAME/hirelens-ai.git
   cd hirelens-ai
   ```

2. **Install Dependencies**:
   ```bash
   npm install
   ```

3. **Environment Variables**:
   Create a `.env` file and add your Gemini API Key:
   ```env
   GEMINI_API_KEY=your_api_key_here
   ```

4. **Run Development Server**:
   ```bash
   npm run dev
   ```

5. **Build for Production**:
   ```bash
   npm run build
   npm start
   ```
